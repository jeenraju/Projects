# In-Network Intrusion Detection Using Neural Networks and P4 Programming Language

### Description:

Intrusion detection mechanisms are vital for protecting information systems from widespread attacks by recognizing anomalous
 network traffic. Because of the high variability of such anomalies and the inherent tradeoff between security and availability
  resulting from its filtering, this has become one of the classical and most successful applications of machine learning (ML)
   classifiers [1]. Programmable switches enable the deployment of machine learning models inside the network allowing us 
   to perform accurate intrusion detection at line rate.

In this project, students design and train a neural network model for this and then deploy the trained model in
 emulated network switches using the P4 programming language [2]. 
 P4 (Programming Protocol-independent Packet Processors) is an open-source domain-specific language that enables 
 writing custom protocols on programable network devices yielding high performance.

### Tasks:

1.	Understand programmable switch architecture and P4 programming language.
2.	Design and train a neural network and measure its accuracy on a provided dataset.
3.	Deploy the ML model on emulated (software) switches to detect intrusions in the switches.

### Requirements:

Fundamental knowledge of machine learning and network protocols.
